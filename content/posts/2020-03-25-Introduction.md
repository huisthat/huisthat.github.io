# About the domain name

This is a post dedicated to Zhi Ying, who helped me brainstorm for my domain name.

I wanted to have a domain name which did not include my name, and could describe me, so I wanted to have one that is the[adjective].me

At first we tried finding adjectives starting with L so that it would be an alliteration with my name, but we realised that most had a negative connotation, such as lazy, lame, lavish etc. Hence that idea is out of the window.

Zhi Ying then suggested emo or moody, which did not exactly describe me, yet it sounded compatible with the format the[adjective].me and I was about to go with it, when she suggested another one.

thetilted.me

I believe that I get tilted really easily, so it was really appropriate and there you go, my domain name currently!
