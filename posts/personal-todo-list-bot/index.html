Introduction

This was an individual project that is making a bot that keep tracks of the todos, deadlines and events. For deadlines and events, there will be a date and time tagged to it. It is implemented using Java, and the GUI is made using JavaFx. The jar file can be found here!

Challenges

One of the biggest challenges was implementing the GUI using JavaFx. I was initially really confused by the stage, scene etc, but after a few days of mulling over it, I can finally figure out how things work and get them to work as I wanted them to. I can still remember I was doing the UI late into the night one day, around 3-4am, and when the program worked as I wanted it to, the sense of happiness was immeasurable.
